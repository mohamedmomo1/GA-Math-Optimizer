# Multi-Function Genetic Algorithm (GA) Optimizer

A Python-based GUI application that uses **Genetic Algorithms** to find the global minimum of various mathematical optimization functions. The tool provides a real-time visualization of the fitness improvement over generations.

##  Features
* **Interactive GUI:** Built with `Tkinter` for easy parameter tuning.
* **Multiple Benchmarks:** Supports optimization for:
    * Sphere Function
    * Sum of Squares Function
    * Booth Function (2D)
    * Beale Function (2D)
    * Matyas Function (2D)
* **Real-time Plotting:** Visualizes the convergence history using `Matplotlib`.
* **Customizable Parameters:** Adjust Population Size, Mutation Rate, Generations, and Bounds.
